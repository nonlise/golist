# Todo List REST API

REST API сервис для управления списком задач. Написан на Go с использованием in-memory хранилища.

## Функциональность

- Создание задачи (требует заголовок и описание)
- Получение задачи по заголовку
- Получение всех задач
- Получение невыполненных задач
- Отметка задачи как выполненной/невыполненной
- Удаление задачи

## API Endpoints

| Метод   | Путь                      | Описание                        |
|---------|---------------------------|---------------------------------|
| POST    | /tasks                    | Создать задачу                  |
| GET     | /tasks/{title}            | Получить задачу по заголовку    |
| GET     | /tasks                    | Получить все задачи             |
| GET     | /tasks?completed=false    | Получить невыполненные задачи   |
| PATCH   | /tasks/{title}            | Изменить статус задачи          |
| DELETE  | /tasks/{title}            | Удалить задачу                  |

## Форматы запросов и ответов

### Создание задачи (POST /tasks)
Запрос:
{
    "title": "Купить молоко",
    "description": "Купить 2 литра молока в магазине"
}

Ответ (201 Created):
{
    "Title": "Купить молоко",
    "Description": "Купить 2 литра молока в магазине",
    "Completed": false,
    "CreatedAt": "2024-01-15T10:30:00Z",
    "CompletedAt": null
}

### Изменение статуса (PATCH /tasks/{title})
Запрос:
{
    "complete": true
}

### Ответ при ошибке
{
    "Message": "task not found",
    "Time": "2024-01-15T10:30:00Z"
}

## Запуск

### Локальный запуск
go mod download
go run main.go

Сервер запустится на порту 9091

### Docker
docker build -t todo-api .
docker run -p 9091:9091 todo-api

## Коды ответа

- 200 OK - успешный GET/PATCH запрос
- 201 Created - задача создана
- 204 No Content - задача удалена
- 400 Bad Request - неверный формат запроса
- 404 Not Found - задача не найдена
- 409 Conflict - задача с таким заголовком уже существует
- 500 Internal Server Error - внутренняя ошибка сервера
